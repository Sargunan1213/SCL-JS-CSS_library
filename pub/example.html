Step 1: Pick one of the existing animations:

1 - No Animation

2 - Opacity

3 - Flip

Step 2: Pick from the faces below (Note: Flip animations require two faces):

Face 1

Face 2

Face 3

Face 4

Step 3: Use the make card function with the arguments as ordered below.



1. Name of the empty HTML element to add the card to. If there are multiple elements with the same name it will chooses the first one.Be wary that the card will occupy the full element.

2. The Card number you chose in step 1.

3. the face number(s) you chose in step 2. If the card uses a flip animation pass in a list of two faces.

4. Array of parameters for the chosen face. If the card uses a flip pass in an array of two elements each of which is an array of parameters as required by the corresponding face.



Sample calls:

Flip animation: makeCard("name", 3, [face_number1, face_number2], [[arguments_to_face_1],[[arguments_to_face_1]])

Other animation: makeCard("name", card_number, face_number, [arguments_to_face])



makeExpand

Use the makeExpand function with the arguments as listed below.

1. Name of the empty html element to add the card to.Be wary that the card will expand to the height needed.

2. An array of two strings, the former is used as the title and the latter as the hidden text.

Sample call:

makeExpand("name", ["Sargunan", "I am a professional developer"])



carouselDots

Please read the getting strated to find the optional arguments, faces and autoplay details. Use the arguments as listed below.

1. Name of the empty HTML element to add the card to.Be wary that the card will occupy the full element.

2. An array of face numbers that the carousel will contain.

3. An array of arrays each of which contains the arguments to the corresponding face.
