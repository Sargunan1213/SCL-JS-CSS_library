SCL Library

The future of cards, sliders and Carousel wheels is here. All it takes is a links and a script. Have a look at the getting started maual for more information on faces, card numbers and optional parameters.

Getting Started

The first step to use the library is adding the script and link tags into the code.

<link rel="stylesheet" type="text/css" href="./css/scl.css"/>

<script defer type="text/javascript" src="./js/scl.js"></script>

Secondly, initialise a new card variable in your javascript file.

const card = new Card();

card.makeCard("card_name", 1, 3, ["image_link","TITLE","SUB-TITLE"]);

Lastly, users using face numbers 2 and 4 must import the Font awesome 4.7 library for the icons.

Other animation: card.makeCard("name", card_number, face_number, [arguments_to_face])

Optional Arguments:

This is an object of key value pairs passes after all required arguments. Passing this object is OPTIONAL.

Face 1: Keys: background, color. Background controls the background of text in the card while color changes text color.

card.makeCard("name", card_number, 1, [arguments_to_face], {background: "white", color: "black"})

Face 2: Keys: title, subtitle, iconcolor, opacity. Title and subtitle control the colors of the title and subtitle respectively. The iconcolor changes the colour of the icon and the opacity key is used to define a layer of opacity on the background image alone.

card.makeCard("name", card_number, 2, [arguments_to_face], {opacity: 0.7, title: "black", subtitle: "black", iconcolor: "blue"})

Face 3: Keys: title, subtitle, opacity. Title and subtitle control the colors of the title and subtitle respectively. The opacity key is used to define a layer of opacity on the background image alone.

card.makeCard("name", card_number, 3, [arguments_to_face], {opacity: 0.7, title: "black", subtitle: "black"})

Face 4: Keys: title, iconcolor, opacity. Title controls the colors of the title. The opacity key is used to define a layer of opacity on the background video alone, while the iconcolor sets the color of the icon.

card.makeCard("name", card_number, 4, [arguments_to_face], {opacity: 0.7, title: "black", iconcolor: "blue"})



makeExpand

Use the makeExpand function with the arguments as listed below.

1. Name of the empty html element to add the card to.Be wary that the card will expand to the height needed.

2. An array of two strings, the former is used as the title and the latter as the hidden text.

Sample call:

card.makeExpand("name", ["SCL", "Card Library"])





carouselDots

Please read the arguments as listed below.

1. Name of the empty HTML element to add the card to.Be wary that the card will occupy the full element.

2. An array of face numbers that the carousel will contain.

3. An array of arrays each of which contains the arguments to the corresponding face.

4. An object containing the optional arguments pertaining to the carosel (OPTIONAL but pass an empty object if you want to use the next argument).

5. An object of keys being the face numbers and values being an object containing the optional arguments to that face. Look at step 3 on makeCard (OPTIONAL).

card.carouselDots("name", [1,2,3,4], [[arguments],[arguments],[arguments],[arguments]])

We first take a look at what each of the control buttons do and how they work. Note these optional arguments are part of the object passed in argument 4 and are OPTIONAL to pass.

The titular function produces a carousel from the faces listed in step 2 of makeCard with only dots to navigate.

The number of dots cap at 5 but still provide the user the ability to navigate all the cards. The dots will adapt and reposition themselves to allow the user access to other cards.

Additionally, the user can add an optional argument, namely buttons, to provide arrow buttons that help scroll the carousel too. These buttons work on a circular basis and move the carosel from end to end if the user hits the right button at the rightend or left button at the left end.

Lastly, the user can also the other optional argument, namely numbers, to provide first and last buttons to move to the start and end of the carousel respectively.

card.carouselDots("name", [1,2,3,4], [[arguments],[arguments],[arguments],[arguments]], {buttons: true}, {1: {background: "white"},...})



Expandable

Use the Expandable function with the arguments as listed below.

1. Name of the empty html element to add the card to.The element must only be set with its height and width as needed for the button to expand. The element will expand beyond the div as a normal navbar does so do not contain the overflow on this element.

2. An array containing arrays of two strings, the former is used as the title and the latter as the hidden text in the order passed in.

Sample call:

card.makeExpand("name", [["SCL", "Card Library"], ["Sargunan", "Web Developer"]])
