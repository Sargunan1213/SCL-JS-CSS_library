SCL Library

Easily add customisable cards, sliders and Carousel wheels to your Webpage. All it takes is two tags.

Step 1: Download the two files here:

Step 2: Add the following tags listed below to <head> in your code modifying the links.
<link rel="stylesheet" type="text/css" href="path/to//scl.css" />
 <script type="text/javascript" src="scl.js"></script>

Step 3: In your JavaScript file create a new instance of Card as below:
const card = new Card();

Step 4: Access the built in functions using the instanciated variable,
s.makeCardAFlip(....);

Step 5: Enjoy using the functions to decorate your webpage.

ards

makeANone(ElementName, ImageUrl, Text)
The function helps create the card shown on the right with hover animations on the image. The first argument to this function is the name of the div element which should hold the card. Note that the height and width properties of the div element must be set and the card will occupy the full div.The second argument is the url of the image to add and the third argument is the text to be added to the card.

makeAOpacity(ElementName, BackgroundImage, ProfileImage, Name, ShortDetails, Links)
The function helps create the card shown on the left with hover animations to change the opacity. The first argument to this function is the name of the div element which should hold the card. Note that the height and width properties of the div element must be set and the card will occupy the full div.The next two arguments are url's for the background and profile images respectively. The next two arguments are for the title text and the short description.The last argument is a list of links to add to the icons.

makeAFlip(ElementName, BackgroundImage, ProfileImage, Name, ShortDetails, Links)
The function helps create a two sided card shown on the right. The first argument to this function is the name of the div element which should hold the card. Note that the height and width properties of the div element must be set and the card will occupy the full div.The next two arguments are url's for the background image(on front side) and the profile image (on the back). The next two arguments are for the title text and the short description. The last argument is an array of 6 links for the icons.

liders

makeSliderOne(ElementName, ListOfImages)
The function helps create the slider of images on the right. The first argument to this function is the name of the div element which should hold the card. Note that the height and width properties of the div element must be set and the card will occupy the full div. The next argument is a list of images to add to the slider.
Note: The slider is set up to move to the first element if the user tries going right on the last element and vice versa.
